# AI Backend Studio ⚡️

ブラウザ完結で動作する、Cloudflare Workersに特化したAI駆動のバックエンド（API）開発スタジオです。
AIとの対話による要件定義から、コードの自動生成、ブラウザ上でのローカルテスト、そしてCloudflare環境へのデプロイまで、API開発に必要なすべてのプロセスを一気通貫で行うことができます。

## 🌟 概要・主な特徴

- **オールインワン**: 要件定義、コーディング、テスト、デプロイが1つの画面で完結します。
- **マルチAIモデル対応**: Gemini 2.5 Flash、Claude 3.5 Sonnet、GPT-3.5 Turboから選択して対話・コード生成が可能です。
- **ブラウザ内ローカルテスト**: Web Workersを利用したサンドボックス環境により、外部サーバーを立てることなくブラウザ上で直接Workerコードのテスト（シミュレーション）が可能です。
- **PWA & レスポンシブ**: PC、スマートフォン、タブレットのどの端末からでも快適に操作できるモダンなUIを採用しています。
- **完全サーバーレス**: バックエンドサーバーを持たず、設定やプロジェクトデータはすべてブラウザのローカルストレージ（IndexedDB）に安全に保存されます。

## ✨ 主な機能一覧

### 💬 要件定義 (チャット)
- 熟練のWebエンジニアとして振る舞うAIと対話しながら、作りたいアプリの要件を詰めることができます。
- 要件が固まると「確定した要件からコードを生成する」ボタンが出現し、一瞬でCloudflare Workerのコード（ES Modules形式）を生成します。

### 💻 コード編集
- 本格的なコードエディタ（Ace Editor）を内蔵。
- 複数ファイルの管理が可能で、シンタックスハイライトや自動保存機能（バックグラウンドで2秒ごとに保存）を備えています。

### 🧪 テスト (ローカルシミュレーション)
- 現在のコードに対して、メソッド（GET, POSTなど）、パス、ヘッダー、ボディを指定して擬似的にリクエストを送信できます。
- Web Worker内でコードを動的にインポートして実行するため、安全かつ高速にAPIの動作確認・デバッグが可能です。

### 🚀 デプロイ
- Cloudflare APIを利用し、ブラウザから直接Cloudflare Workers環境へスクリプトをデプロイできます。
- ターミナル風のUIでデプロイの進行状況や成功/エラーログを確認できます。

### ⚙️ 環境設定
- 使用するAIモデルのAPIキー（Gemini, Claude, OpenAI）の設定。
- Cloudflareのデプロイに必要な認証情報（Account ID, API Token）の設定。
- AIの振る舞いを細かく調整できるシステムプロンプトや、コード生成要件のカスタマイズ。

### 📦 プロジェクト管理
- プロジェクト全体（複数ファイル・チャット履歴・設定）のZIPエクスポートとインポート機能。
- 直近5件の自動バックアップ機能により、誤ってコードを消してしまった場合でも履歴から復元可能です。

## 🛠 使用技術・ライブラリ

- **フロントエンド**: HTML5, Vanilla JavaScript, CSS3
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/) (CDN)
- **コードエディタ**: [Ace Editor](https://ace.c9.io/) (CDN)
- **ストレージ・データ管理**: [localForage](https://localforage.github.io/localForage/) (IndexedDBラッパー / CDN)
- **ファイル圧縮/解凍**: [JSZip](https://stuk.github.io/jszip/) (CDN)
- **コアAPI**: Web Workers API (AI非同期通信・コード実行サンドボックス), Blob URL, Fetch API, PWA (Manifest)

## 🚀 セットアップ・使い方

本アプリケーションはビルドプロセスを必要としません。以下の手順ですぐに使用を開始できます。

### 1. 起動方法
提供された `index.html` をお使いのモダンブラウザ（Chrome, Edge, Safari推奨）で開くだけで起動します。
※ セキュリティの都合上（Web WorkerでのESM動的インポートなど）、VS Codeの「Live Server」などのローカルサーバー経由での起動を推奨します。

### 2. 初期設定
1. アプリが起動したら、左側（モバイルの場合は下部）のメニューから **「⚙️ 設定」** タブを開きます。
2. 以下の必要な情報を入力し、「設定を端末に保存」をクリックします。
   - **AI API Keys**: Gemini, Claude, OpenAI のいずれか（または全て）のAPIキー
   - **Cloudflare 認証情報**: Cloudflareの `Account ID` と `API Token`（※Tokenは「Edit Workers」権限が必要）

### 3. 開発フロー
1. **要件定義**: 「💬 要件定義」タブで作りたいAPIの要件をAIに伝えます。
2. **コード生成**: 対話が進むと表示される「⚡️ 確定した要件からコードを生成する」ボタンをクリックします。
3. **編集**: 「💻 コード編集」タブに自動的に移動するので、生成されたコードを確認・必要に応じて修正します。
4. **テスト**: 「🧪 テスト」タブで、リクエスト情報（URLパスやJSONボディ）を入力して「現在のコードでテスト実行」をクリックし、期待通りのレスポンスが返ってくるか確認します。
5. **デプロイ**: テストが完了したら「🚀 デプロイ」タブに移動し、任意のWorker名を入力して「Cloudflareへデプロイ」をクリックします。

### 4. データの持ち運び
右上の「📦」ボタン（PC版は左下「出力」）から、現在のプロジェクト状態をZIPファイルとしてダウンロードできます。別のPCやブラウザで作業を再開したい場合は、隣の「📂」ボタン（インポート）からZIPファイルを読み込ませてください。
